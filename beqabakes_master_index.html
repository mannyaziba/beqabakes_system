Quick reference

Brand mode decision table

Express
Urgency + sales. Lead with action. Price visible. Max 2 lines before CTA. Single WA link. Rosewood (#D16681) CTA. Used for Monday posts, offer broadcasts, same-day pushes.
"Pick. Pay. Done."
Celebration
Warmth + gift moment. Recipient-first. Slightly longer, still ends in action. Cake-Card Wedge visual. Pink (#F58BA6) accent. Used for Wednesday posts, Reels, packaging.
"Every box feels like a celebration."
Retention
Soft + relational. Acknowledge the gift. Low pressure, one action only. BACK500 code. Blush palette. Used for Friday posts, Day 7 nudge, loyalty mechanic, packaging insert.
"Hope they loved it."
Mode decision tree (3 questions): 1. Has this customer already ordered? → Retention. 2. Is this driving an order right now? → Express. 3. Does it show a gift moment or emotion? → Celebration. If none → rethink, don't publish.

Quick reference

The non-negotiable laws

System law

"We don't create content. We trigger gifting decisions."

Platform law

"Instagram creates desire. WhatsApp closes it. One brief builds both."

Conversion law

"Remove thinking → show options → ask 'which one?' Every extra step is a lost order."

Production rules

What is true every single week

  • 3 posts minimum. Never fewer. Mon (Express) · Wed (Celebration) · Fri (Proof/Retention).
  • Every post starts with a completed SIA+ brief with mode assigned before any copy or visual work.
  • Every post passes the 6-point pre-output checklist before publishing. No checklist pass = no publish.
  • Every delivery triggers Script 03 (Day 0 photo prompt) and schedules Script 04 (Day 7 nudge).
  • All five metrics are logged in Notion every Friday. No exceptions.
  • The Proof bank is checked before any Friday post is created. If bank < 4 items, flag immediately.

Absolute limits

Things that never happen

  • No post publishes with ₦[X] — a missing price destroys trust faster than a missing image.
  • No CTA destination other than WhatsApp — directly or via the bio link router (which also closes to WhatsApp on every path).
  • Rosewood (#D16681) is the only CTA color. Primary Pink (#F58BA6) is never used on buttons.
  • Merienda font is used on physical gift tags and note cards only — never in digital UI or captions.
  • Facebook never runs a Sunday post. No content day is added without a SIA+ brief and checklist.
  • No paid ads before threshold: 15+ WA order initiations/week × 4 consecutive weeks.
  • No separate content systems per platform. One SIA+ brief produces all three platform outputs.
  • Quick Order product names are canonical: Quick Birthday Cake, Send Love (Bento), Mini Celebration Box, Just Because Cake, Cupcake Box. No abbreviations, no invented names — these appear identically in posts, scripts, Paystack links, and the tracking sheet.
